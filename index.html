Starting with HTML

April 6, 2025

Today I began learning HTML, the foundation of all websites. I learned how to create headings, paragraphs, and lists. It's exciting to see how these simple tags can create structure for web pages.

I'm still getting comfortable with how elements are displayed on the page. Some elements take up the whole width while others only take as much space as they need.

Discovering CSS

April 10, 2025

Four days after starting HTML, I began learning CSS. It's amazing how CSS can change the appearance of my HTML elements. I learned how to change colors, fonts, and spacing.

I'm enjoying experimenting with different styles and seeing how they affect my web pages. There's so much to learn, but each small success feels rewarding!
